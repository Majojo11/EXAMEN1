3. Tipos de aplicaciones móviles (10 pts)
Explica brevemente:
    Aplicaciones nativas: Son las aplicaciones que estan hechas para un SO especifico, utilizan los mismos lenguajes (como IOS con SWIFT), tienen mejor rendimiento en el sistema operartivo y aprovechan todo el potencial del hardware porque son echos para esos dispositivos

    Aplicaciones híbridas: Son aplicaciones creadas con lenguajes como HTML, CSS, JavaScrip, etc. 
        Nos permiten usar el mismo código para diferentes plataformas, pero puede que el rendimiento sea menor que las nativas.

    Aplicaciones web: Son aplicaciones que se ejecutan en el navegador, no en el dispositivo, el dispositivo tiene que contar con internet o no funcionara y no necesita una instalación previa.

¿Cuál es la diferencia principal entre ellas?
    las aplicaciones nativas tienen un mejor rendimiento y puede aprovechar todo el potencial de este, mientas que las hibridas tendrán un rendimeinto menor pero es adaptable a los diferentes SO, por lo que podemos reutilizar el codigo y las aplicaciones web funcionan desde el navegador en cualquier SO y no necesita ninguna instalación
